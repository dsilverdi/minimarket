# Mini Market

set up the whole server module for mini marketplace platform

## Technical Constrainst
* Go 1.14
* MySQL 5.7
* Redis 4.0+

## Depedencies
* Docker
* Docker Compose

## How to run
to run this service you should have Docker installed beforehand. 

build service
```sh-session
$ docker compose build
```

to run this service
```sh-session
$ docker compose up
```
